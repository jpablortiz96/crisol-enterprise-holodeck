# Azure Resource Group Screenshot

Capture Azure resource group showing Container Apps, Container Apps
Environment, Azure AI Search, Foundry/AIServices resource, Log Analytics, and
Container Registry.

Recommended capture:

1. Open the production resource group in Azure Portal.
2. Keep resource names, types, locations, and status visible.
3. Hide subscription IDs, tenant IDs, tags containing private data, costs,
   activity logs, access keys, connection strings, and identity details.
4. Save the image as `azure-resource-group.png` in this directory.
