# Manager Readiness Rubric RUBRIC-MGR-013

Synthetic demonstration document.

## Purpose

Use this rubric to review whether a learner is ready for role-based incident practice.

## Signals

- Can explain the affected system chain without guessing.
- Can name missing evidence and request it clearly.
- Can balance customer impact, data safety, and restoration speed.
- Can communicate uncertainty without hiding risk.
- Can identify when certification study should precede scenario escalation.

## Scoring

Use a 0 to 100 practice score. Scores under 60 indicate foundational practice. Scores from 60 to 79 indicate guided readiness. Scores of 80 or higher indicate advanced scenario readiness.
